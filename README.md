# Economy
# Este site educativo apresenta conceitos de Economia Doméstica e Finanças Pessoais integrando Matemática, Geografia, Sociologia e Português.
Contém um simulador de orçamento mensal (entrada de receitas/despesas e cálculo automático), dicas de consumo consciente, infográficos e um glossário com +10 termos financeiros.
O objetivo é promover organização financeira, pensamento crítico e alfabetização econômica por meio de uma interface simples e acessível.
Todo o código é em HTML, CSS e JavaScript puro — sem frameworks externos — e o site está preparado para publicação estática no Netlify.
